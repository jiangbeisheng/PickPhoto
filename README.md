## 开发背景

这是一个基于 iOS 的照片读取与导入工具，主要用于解决摄影用户在使用读卡器向 iPhone 导入相机照片时的效率问题。

目前 iPhone 原生方案在实际使用过程中存在很多体验上的痛点。

### 方式一：使用 iPhone「文件 App」读取内存卡

通过文件 App 浏览 SD 卡或 USB 存储设备时：

* 照片缩略图过小，几乎无法高效选图
* 很难快速筛选废片
* 通常只能按时间“一梭哈”全部导入
* 导入后无法标记哪些照片已经导入过
* 如果中途导入失败，很难继续断点导入
* 需要手动删除重复照片
* 甚至需要人工记录照片文件名编号，分批导入

整个过程非常低效且容易混乱。

---

### 方式二：使用 iPhone 相册原生 Import 功能

iPhone 相册自带的 Import 功能也存在一些问题：

* 会扫描整个内存卡
* 无法指定某个目录导入
* 无法按拍摄日期或文件夹快速筛选
* 默认升序排列
* 最新拍摄的照片在最底部
* 当照片数量达到几千张时，需要不停下滑查找最新照片

对于经常拍摄 RAW、大量连拍、或者多目录管理照片的用户来说体验并不好。

---

## 这个 App 想解决什么问题

这个项目希望提供一种更适合摄影用户的照片导入体验：

* 支持直接浏览 USB / SD 卡目录
* 支持大图预览与快速选图
* 支持按目录导入
* 支持倒序查看最新照片
* 支持标记已导入照片
* 支持断点续导
* 减少重复导入与手动清理
* 更适合大量照片管理场景

这是一个主要通过 vibe coding + AI 辅助开发的 iOS 实验项目。
